HOWTO: https://spring.io/guides/gs/serving-web-content/#initial


If you are using Gradle, you can run the application using ./gradlew bootRun. 
Or you can build the JAR file using 		./gradlew build. 
Then you can run the JAR file:			java -jar build/libs/gs-serving-web-content-0.1.0.jar

Now that the web site is running, visit http://localhost:8080/
